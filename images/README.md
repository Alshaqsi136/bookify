# Images Directory

This directory should contain the following images for the website:

## Required Images

1. **hero-bg.jpg** - Hero background image for the home page (referenced in CSS)
2. **hotel-image.jpg** - Sample hotel image (can be used in multiple pages)
3. **room-images/** - Directory for room type images
4. **hotel-logos/** - Directory for hotel logos (optional)

## Image Guidelines

- Keep total folder size under 100MB
- Use optimized image formats (JPEG for photos, PNG for graphics)
- Recommended dimensions:
  - Hero images: 1920x1080px
  - Hotel images: 800x600px
  - Room images: 600x400px
- Compress images before adding to reduce file size

## Placeholder Images

For development purposes, you can use placeholder services like:
- https://via.placeholder.com/800x600
- https://picsum.photos/800/600

Or create your own images using image editing software.
